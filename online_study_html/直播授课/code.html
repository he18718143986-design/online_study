Theorem 1.1: Linear Independence

If S = {v₁, v₂, ..., vₙ} is a set of vectors in V, then the vector equation:

c₁v₁ + c₂v₂ + ... + cₙvₙ = 0

has only the trivial solution c₁ = c₂ = ... = cₙ = 0.
