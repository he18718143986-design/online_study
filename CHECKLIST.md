## 验证命令序列

复制以下命令到终端执行完整验证：

```bash
# 完整验证流程
echo "=== 1. 安装依赖 ===" && \
pnpm install && \
echo "=== 2. 类型检查 ===" && \
pnpm typecheck && \
echo "=== 3. 代码检查 ===" && \
pnpm lint && \
echo "=== 4. 单元测试 ===" && \
pnpm test && \
echo "=== 5. 生产构建 ===" && \
pnpm build && \
echo "=== 全部验证通过 ==="
```

E2E 测试（需要单独运行）：

```bash
# 安装 Playwright 浏览器（首次）
npx playwright install

# 运行 E2E 测试
pnpm test:e2e
```
